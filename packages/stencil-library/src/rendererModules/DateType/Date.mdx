# Date

The Date renderer detects ISO 8601 date strings and renders them in a user-friendly, locale-aware format. Dates are simple types with no expandable details.

## Supported Formats

- Full ISO 8601: `2022-11-11T08:01:20.557+00:00`
- Date only: `2024-06-15`
- Date with time: `2024-06-15T09:30:00.000+02:00`

## Settings

No additional settings are required. The renderer uses the browser's locale defaults for formatting.
