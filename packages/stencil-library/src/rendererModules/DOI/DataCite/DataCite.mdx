# DataCite DOI Renderer

The DataCite DOI renderer resolves DOIs by fetching metadata directly from the [DataCite API](https://api.datacite.org/). It is designed to work exclusively with DataCite DOIs and provides detailed metadata extraction.

## Settings

- **`citationStyle`** (`string`, default: `APA`): The citation style for the preview. Options: `APA`, `Chicago`, `IEEE`, `Harvard`, `Anglia Ruskin`
- **`ttl`** (`number`, default: `86400000`): Time-to-live in milliseconds for cached metadata

## Usage

```html
<pid-component value="10.5445/IR/1000188026" renderers='["DataCiteDOIType"]'></pid-component>
```

### With Settings

```html
<pid-component
  value="10.5445/IR/1000188026"
  renderers='["DataCiteDOIType"]'
  settings='[{"type":"DataCiteDOIType","values":[{"name":"citationStyle","value":"IEEE"}]}]'>
</pid-component>
```

## Features

- **ORCID Extraction**: Automatically extracts ORCID identifiers from creator metadata
- **ROR Extraction**: Extracts ROR (Research Organization Registry) identifiers from affiliation data
- **Metadata Caching**: Uses cached API responses to improve performance
- **Multiple Resource Types**: Supports journal articles, software, RFCs, preprints, and more
