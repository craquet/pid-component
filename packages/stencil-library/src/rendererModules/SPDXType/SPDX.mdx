# SPDX License

The SPDX renderer recognizes SPDX license identifiers (e.g., `Apache-2.0`) or URLs. It validates the license against the SPDX database/API and provides links to the full license text and details.

## Settings

- **`ttl`** (`number`, default: `86400000`): Time-to-live for cached license data

## Usage

```html
<pid-component value="https://spdx.org/licenses/Apache-2.0"></pid-component>
```

### Bare Identifier

```html
<pid-component value="Apache-2.0"></pid-component>
```

### With Settings

```html
<pid-component
  value="Apache-2.0"
  settings='[{"type":"SPDXType","values":[{"name":"ttl","value":3600000}]}]'>
</pid-component>
```
