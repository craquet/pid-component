# ROR (Research Organization Registry)

The ROR renderer resolves ROR IDs to identify research organizations. It fetches organization details such as name, acronyms, location, and relationships (parent/child organizations) from the ROR API.

## Settings

- **`ttl`** (`number`, default: `86400000`): Time-to-live for cached organization data

## Usage

```html
<pid-component value="https://ror.org/04t3en479"></pid-component>
```

### With Settings

```html
<pid-component
  value="https://ror.org/04t3en479"
  settings='[{"type":"RORType","values":[{"name":"ttl","value":3600000}]}]'>
</pid-component>
```
