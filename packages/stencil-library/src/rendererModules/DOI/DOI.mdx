# DOI (Digital Object Identifier)

The DOI renderer resolves DOIs (e.g., `10.1000/xyz`) to fetch metadata from providers like DataCite or CrossRef. It renders a formatted citation preview and allows expanding to view detailed metadata (authors, publication dates, licenses, etc.).

DOIs are resolved via `doi.org` which automatically redirects to the appropriate provider (DataCite or CrossRef). The metadata source is determined by the structure of the returned data.

## Settings

- **`citationStyle`** (`string`, default: `APA`): The citation style for the preview. Options: `APA`, `Chicago`, `IEEE`, `Harvard`, `Anglia Ruskin`
- **`ttl`** (`number`, default: `86400000`): Time-to-live in milliseconds for cached metadata

## Usage

```html
<pid-component value="10.5445/IR/1000188026"></pid-component>
```

### With Settings

```html
<pid-component
  value="10.5445/IR/1000188026"
  settings='[{"type":"DOIType","values":[{"name":"citationStyle","value":"IEEE"}]}]'>
</pid-component>
```
